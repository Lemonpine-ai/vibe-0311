# VoiMeow 웹소개페이지 - 구현 참고사항

## ✅ 현재 구현된 기능
- VoiMeow 로고 및 네비게이션 메뉴 (상단 고정)
- 행복한 고양이 배경 이미지 슬라이드 (페이드 전환)
- App Store / Google Play 다운로드 버튼 UI
- **회사소개**: 고양이와 함께하는 행복한 삶, VoiMeow 소개, 깜빡이는 로고
- **로그인**: ID test / 비밀번호 test 로 로그인
- **로그인 후**: 마이페이지, 로그아웃, 사이드바(구독서비스관리, 고양이관리, 실시간리포트, 가족연결설정)
- **사용방법**: 앱 설치→로그인→고양이설정→동영상 영역 (실제 동영상 URL 교체 필요)
- **커뮤니티**: 묘종별(A~H), 연령별(1년미만/9세이상), 질병별(A~E), 기타1·기타2 슬라이드
- 커뮤니티 대시보드: 큰그림/글씨만 모드, 제목/내용/작성자 검색, 글쓰기
- 푸터 (주소, 연락처, 챗봇/원격지원/파트너쉽)

---

## 🔧 추후 연결이 필요한 부분

### 1. 사용방법 동영상
- `index.html`의 `.usage-video-placeholder` 영역을 실제 동영상 iframe으로 교체
- 예: `<iframe src="https://www.youtube.com/embed/영상ID" ...></iframe>`

### 2. 스토어 링크
- App Store: `https://apps.apple.com/app/id[앱ID]`
- Google Play: `https://play.google.com/store/apps/details?id=[패키지명]`

### 3. 로그인 (실제 서비스)
- 현재 test/test로 데모 로그인만 가능
- 백엔드 인증(세션, JWT) 연동 필요

### 4. 챗봇 / 원격지원 / 파트너쉽
- 각 서비스별 외부 연동 필요

### 5. 공지사항, 이벤트, 고객센터
- 현재 플레이스홀더만 표시됨
